# Kelp Profit Calculator

A local Flask web app for estimating:
- blaze rod cost
- blaze rod stacks per smoker
- total rod stacks
- processed kelp
- dried kelp blocks and stacks
- gross revenue
- net profit
- whether the bottleneck is farm growth or smoker throughput

## Run locally

1. Open a terminal in this folder.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Start the app:

```bash
python app.py
```

4. Open:
- `http://127.0.0.1:5000` on your PC
- `http://YOUR_LOCAL_IP:5000` on another device on your network
